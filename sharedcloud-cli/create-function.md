# Create function

It creates a new function.

#### Example:

```
sharedcloud function create --name helloWorld \
                            --image-uuid e74439ae-ffb2-407a-97c2-f7cc925173e8 \
                            --file helloworld.py

```

#### Arguments:

* **name:** name of the function
* **image-uuid**: uuid of the image
* **file**: path to the file
